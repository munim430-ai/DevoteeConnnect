# DevoteeConnect – Landing Page

This repository contains a simple static landing page for **DevoteeConnect**, a premium digital sanctuary for modern devotees. The site introduces the core features of the DevoteeConnect app, including the Sadhana tracker, Sanga explorer, AI Wisdom Guide, Ekadashi portal, Devotee feed, and an interactive **Sadhana Rhythm Finder**.

## Contents

- `index.html` – The main HTML file that structures the landing page.
- `styles.css` – Custom styles that apply the spiritual colour palette and layout.
- `script.js` – A lightweight script powering the Sadhana Rhythm Finder quiz.
- `.nojekyll` – Disables Jekyll processing on GitHub Pages.
- `README.md` – This file.

## Building & Deployment

These files are ready to be deployed directly to GitHub Pages. To publish this site:

1. Upload the files in this folder to the root of your repository (for example `munim430-ai/DevoteeConnnect`).
2. Commit and push the changes to the `main` branch.
3. In your repository settings, enable **GitHub Pages** and choose the `main` branch as the source with the `/ (root)` directory.
4. Save the settings and wait a few minutes for your site to be published at `https://<username>.github.io/<repository>/`.

## Notes

- This landing page is a lightweight approximation of the full design brief for DevoteeConnect. It uses static placeholders for the map, feed and AI sections and focuses on an elegant aesthetic with dark backgrounds, warm gold accents and simple interactions.
- The interactive **Sadhana Rhythm Finder** quiz does not store or transmit any information; it provides an on-page recommendation only. In the full application this quiz would integrate with backend services to personalise user experiences.
- Feel free to customise the content, colours or layout to better suit your branding and product roadmap.
